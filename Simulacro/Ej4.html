Ejercicio 4

Crea un script que contenga un objeto de un tipo que defina el alumno (puede ser productos, alimentos, alumnos, personas...) después muestra dichos elementros por pantalla, los elementos que introduzcamos deben de almacenarse en memoria local. También tenemos que tener la opción de poder borrar un elemento en específico, cuando se borre el elemento hay que actualizar la memoria local para que dicho elemento se borre. También tienes que introducir una opción para editar los campos, una vez editados debes actualizar los datos existentes en la memoria local.
