Ejercicio 1

Utilizando catalogo.xml, muestra todos aquellos CDs cuya nacionalidad sea UK. Una vez se muestre los elementos, se debe poder borrar aquellos CDs que queramos, cada CD tiene que tener un botón correspondiente para borrarlo. Además, se debe de poder introducir CD, los datos que tiene que introducir usuario son: el título, artista, compañía, precio y año, la nacionalidad es UK. Cada CD tiene que ser un elemento independiente.
