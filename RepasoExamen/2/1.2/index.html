Ejercicio 2 de la relación de ejercicios 2

Crear una web con varios cuadros de texto. La web tendrá una función llamada enfoque que se llamará cuando oscurra el evento onfocus de cada cuadro de texto. Cuando se dispare dicho evento se pondrá el color del fondo del cuadro gris claro y la letra en azul. Hacer que cuando se pierda el enfoque el cuadro tenga el formato original
